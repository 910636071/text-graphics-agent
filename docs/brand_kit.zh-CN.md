# 品牌手册

英文原文：[Brand Kit](./brand_kit.md)。

## 名称

Text Graphics Agent

## 简称

TGA

## 定位

面向一次性子 agent 工作流的语义防火墙。

## 核心句

Models propose. Records decide.

中文口径：

模型负责提案，记录负责裁决。

## 语气

- 精确；
- 研究级；
- 系统导向；
- 反 hype；
- 证据优先。

## 色板

| Token | Hex | 用途 |
| --- | --- | --- |
| Ink | `#05070D` | 背景 |
| Panel | `#0B1020` | 卡片和表面 |
| Cyan | `#00D1FF` | 已验证流程 |
| Violet | `#7C3AED` | 模型智能 |
| Green | `#4ADE80` | accepted records |
| Red | `#F87171` | polluted inputs |
| Slate | `#64748B` | 次要文字 |

## 公开短句

- 面向一次性子 agent 工作流的语义防火墙。
- 模型负责提案，记录负责裁决。
- 不要让最聪明的模型直接成为状态写入者。
- 用户语义是动机，不是证据。
- 生成候选很便宜，接受状态很昂贵。

## 社交预览文案

```text
Text Graphics Agent

Models propose. Records decide.

一种母子 agent 架构：在语义污染进入持久状态前阻断它。
```

## 避免

- AGI 声明；
- 反竞争对手叙事；
- “解决幻觉”；
- “完全安全”；
- 泛泛的“multi-agent platform”话术。
