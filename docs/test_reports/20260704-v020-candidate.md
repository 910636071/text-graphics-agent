Boundary: this is not a playable-game acceptance test. It only confirms the
live model can produce a parseable proposal that passes the current
`AgentProposal` and `ConstraintChecker` gates. The file was not written, the
browser did not run it, and game behavior such as movement, collision, scoring,
restart, and layout was not validated. That full functional test is blocked
until G3 completes write-tool staging and accepted-record commit semantics.

### Local Snake Artifact Functional Check

After the proposal-smoke boundary was clarified, a local Codex-authored
example was saved as `examples/snake_game.html` and tested with headless
Chromium. This is a local artifact acceptance check, not evidence that TGA
child agents can write files.

Observed result:

```json
{
  "ok": true,
  "title": "TGA Snake",
  "canvas": {
    "width": 520,
    "height": 520
  },
  "nonBlankPixels": 230400,
  "scoreAfterEat": 1,
  "lengthAfterEat": 4,
  "selfCollisionGameOver": true,
  "wallCollisionGameOver": true,
  "restartScore": 0,
  "pauseToggle": ["Paused", "Running"]
}
```

Accepted checks:

- page loads as a standalone browser file
- canvas renders nonblank game content
- eating food increments score and grows the snake
- self collision ends the run
- wall collision ends the run
- restart clears `gameOver` and resets score
- pause/resume button updates state
- desktop and mobile layouts have no horizontal overflow

## Remaining Release Boundary

This candidate supports G1 and G2 local claims:

- auditable file-derived evidence provenance
- deterministic clean acceptance metrics with false-positive reporting
- live proposal smoke and live chat smoke, with proposal smoke explicitly
  separated from functional acceptance
- local snake artifact functional check, explicitly separated from TGA
  write-capable child-agent claims

It should not yet claim write-capable agent execution. The next release gate is
G3: write tools must stage effects and commit only after accepted records. The
current patch-preview path is read-only and does not count as write-capable
agent execution.
